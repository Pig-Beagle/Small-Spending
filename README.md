# small-spending-api
지출 관리 커뮤니티

# 🖥️ 프로젝트 소개
지출 내역을 올려 다양한 사람들과 소통하는 서비스 입니다.
<br>
실시간 채팅 형식으로 댓글을 작성할 수 있습니다.
